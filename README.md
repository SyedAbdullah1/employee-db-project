# employee-db-project
The project is built in NodeJS using mongo database. The project is about the real-life problem  Entering the employee info and saving it in database and also it stores the item info and show  the order details that have been placed. 
1-Extract the file 
2- open the folder in vs code
3-Open the terminal 
4-Check that you are in the folder directory
5-write the command npm install
6- after that write the command node server.js
it will start your server , you can open the server by typing the following URL

localhost:3000
OR
localhost:3000/employee

if this port doesnt work open your terminal there you can see the port number

and change the port number with 3000
